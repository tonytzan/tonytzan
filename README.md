## 👨‍💻 About Me

I'm a **Senior Software Engineer** at [CleverCards](https://clevercards.com), based in **Ho Chi Minh City, Vietnam** 🇻🇳.

I specialize in cross-platform mobile development and backend engineering — building products that are fast, reliable, and delightful to use. I care a lot about clean architecture, developer experience, and mentoring the next generation of engineers.

- 🔭 Currently building fintech-grade mobile SDKs and DX tooling at **CleverCards**
- 🌱 Exploring **AI-assisted development workflows** and multi-agent systems
- 🏗️ Passionate about **Flutter**, **clean architecture**, and **scalable APIs**
- 💬 Ask me about mobile engineering, Flutter, or how to level up as a developer
